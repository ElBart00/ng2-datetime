Angular2 datetime component v1.0.0

Dependencies:

  • Bootstrap3 (CSS only)
  • jQuery 2.x
  • Bootstrap-timepicker (JS+CSS) (version included in src files)
  • Bootstrap-datepicker (JS+CSS) (version included in src files)

Default behaviour (plugins default options)

<datetime [(ngModel)]="date"></datetime>

ngModel: {{getDate(date) | date: "medium"}}

Timepicker options (see all)

<datetime [timepicker]="{showMeridian: false, minuteStep: 1}" [(ngModel)]="date2"></datetime>

ngModel: {{getDate(date2) | date: "medium"}}

Datepicker options (see all)

<datetime [datepicker]="{daysOfWeekDisabled: [0,6]}" [(ngModel)]="date3"></datetime>

ngModel: {{getDate(date3) | date: "medium"}}

Empty model (date4 === undefined)

<datetime (ngModel)]="date4"></datetime>

ngModel: {{getDate(date4) | date: "medium"}}
